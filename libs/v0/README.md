# qz-ui

## Install
```shell
npm install qzui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import qzui from 'qzui'

Vue.use(qzui)

// or
import {
  DataTable,
  FormLayout
  // ...
} from 'qzui'

Vue.component(DataTable.name, DataTable)
Vue.component(FormLayout.name, FormLayout)
```
